CityCourier lets you create a courier delivery system with distance-based pricing and Google Maps integration.

== Description ==
Free features:
* Google Maps based address selector
* Distance Matrix calculation for delivery time & pricing
* WooCommerce order generation

Pro version includes courier assignment, live tracking, reports and more.

== Installation ==
1. Upload the plugin zip file via WordPress admin
2. Activate the plugin
3. Configure your Google API key in the settings

== Frequently Asked Questions ==
= Is this plugin free? =
Basic features are free. Pro version adds tracking & assignments.
